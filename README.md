# Family OS

Family OS - семейная CRM/OS для недельного плана, задач дня, статусов детей, дома, наград, библиотеки развития и отчётов.

Главная логика продукта:

```text
недельный план -> задачи дня -> чек-листы Яны и Евы -> отчёты для семьи
```

## Текущий статус

Проект развивается в двух слоях:

- `preview/index.html` - актуальный кликабельный прототип, которым сейчас пользуемся для быстрых решений по интерфейсу.
- `src/` - React + TypeScript + Supabase версия для полноценного приложения с аккаунтами.

Если нужно понять самые свежие продуктовые решения, сначала смотрите `PROJECT_MEMORY.md` и `docs/`.

## Структура

- `preview/index.html` - главный текущий preview.
- `docs/` - документация для продолжения проекта.
- `src/pages` - основные страницы: auth, onboarding, dashboards, задачи, награды, отчеты, дом.
- `src/components` - общий shell, защищенные маршруты и UI-примитивы.
- `src/modules/auth` - состояние Supabase Auth и профиль пользователя.
- `src/modules/family` - создание семьи, подключение по invite code, список детей.
- `src/modules/tasks` - создание задач, отправка выполнения, подтверждение родителем.
- `src/types` - доменные типы и минимальные типы Supabase.
- `supabase/schema.sql` - таблицы, enum-типы и RLS policies.

## Запуск

Быстрый просмотр без сборки:

```bash
python3 -m http.server 4173 --directory preview
```

После этого откройте `http://localhost:4173`.

React-приложение:

```bash
cp .env.example .env.local
npm install
npm run dev
```

В `.env.local` нужно указать `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY`.
Перед первым запуском выполните SQL из `supabase/schema.sql` в Supabase SQL Editor. Он создаст таблицы, RLS policies, trigger профиля после регистрации, функцию `review_task` и Storage bucket `task-reports`.

## MVP

В preview сейчас реализовано:

- реальный недельный план от выбранной даты;
- семейный шаблон дня для Яны и Евы;
- акробатика во вторник и четверг;
- генерация задач дня из недельного плана;
- отдельные чек-листы Яны и Евы;
- статусы Яны и Евы;
- создание задач;
- конкретные награды и создание награды;
- библиотека развития и добавление материалов;
- семейный чат с удалением сообщений;
- отчёты с честным нулевым состоянием.

В React/Supabase версии подготовлено:

- `/register` и `/login` через Supabase Auth.
- Выбор роли: родитель или ребенок.
- `/onboarding`: родитель создает семейное пространство или входит вторым родителем по invite code, ребенок входит по invite code.
- Защищенные маршруты по ролям.
- `/parent/dashboard`: дети, задачи, статусы, просрочки, быстрые отчеты, подтверждение задач.
- `/parent/dashboard`: создание пунктов плана дня для детей.
- `/child/dashboard`: план дня из `daily_plans`, задачи, комментарий к выполнению, загрузка фото-отчета, баллы и награды.
- `/tasks`: создание задач родителем, список задач и комментарии внутри задач.
- `/rewards`: мотивация, уровни, создание и просмотр наград, достижения.
- `/reports`: дневная и недельная статистика.
- `/home`: покупки и домашние обязанности через Supabase, напоминания, быстрые сообщения через `notifications`.
- PWA manifest и service worker через `vite-plugin-pwa`.
